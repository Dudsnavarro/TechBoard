📚 Projeto de Estudo — TechBoard

Projeto desenvolvido para estudo e prática de React.

A aplicação consiste em um quadro de eventos onde é possível visualizar eventos cadastrados e adicionar novos por meio de um formulário. Os eventos são organizados automaticamente por temas.

O objetivo do projeto foi praticar conceitos básicos do React, como componentes, estados, props e renderização dinâmica de listas.
